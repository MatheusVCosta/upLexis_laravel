<h1 align="center">Teste-upLexis</h1>

<h3>Sobre o teste</h3>
<p>Aplicação desenvolvida em Laravel que faz a captura de artigos no blog da upLexis</p>

<h3>O que foi usado</h3>
<ul>
    <li>PHP 7.3</li>
    <li>Laravel 5.7</li>
    <li>MySQL</li>
    <li>AJAX</li>
    <li>Bootstrap</li>
</ul>

<ul>
